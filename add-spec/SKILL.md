# Add Spec

1. Read the user's requirement.
2. Determine which spec doc it belongs to
3. Append or update the relevant section in that doc.
4. Add a corresponding unchecked `[ ]` item to relevant checklist.
5. Do NOT implement the requirement — only update docs and checklist.
